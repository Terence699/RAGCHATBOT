# Project Purpose

This is a Course Materials RAG (Retrieval-Augmented Generation) System that enables users to query course materials and receive intelligent, context-aware responses.

## Key Features
- Semantic search across course documents (PDF, DOCX, TXT)
- AI-powered responses using Anthropic's Claude
- Web interface for user interaction
- Document ingestion from docs/ directory
- ChromaDB vector storage with dual collections (course_catalog and course_content)
- Tool-based search system for enhanced accuracy

## Target Use Case
Educational content management and intelligent querying for course materials.
